Baby.HTML tools

These are some things you can use in Baby.HTML to manage cool things! They're all on Github Gist, so you can find these tools there as well. Simply copy-and-paste these code tidbits into Baby.HTML and you're off!

The current tools are:

  • Cookies Galore, a cookie getter/setter

  • Baby's First Multiplication, a multiplication game
